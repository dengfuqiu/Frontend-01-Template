# Week02 作业
## 写一个正则表达式匹配所有Number直接量
### 二进制
```
/^0[bB][01]+$/
```
### 八进制
```
/^0[oO][0-7]+$/
```
### 十六进制
```
/^0[xX][0-9a-fA-F]+$/
```
### 整数
```
/^[-+]?[0-9]+$/
```
### 指数
```
/^[0-9][eE]([-+]?)[0-9]+$/
```
### 所有 Number直接量的正则
```
/(^0[bB][01]+$)|(^0[oO][0-7]+$)|(^0[xX][0-9a-fA-F]+$)|(^[-+]?[0-9]+$)|(^[0-9][eE]([-+]?)[0-9]+$)/
```
